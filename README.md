
# AI Assistant System (Modular LLM Architecture)

A lightweight AI assistant system with modular backend support and memory handling.

---

## Features

- Conversation memory system
- Prompt orchestration layer
- Swappable backend design (local / API-ready structure)
- CLI chat interface

---

## Architecture

User → Prompt Builder → LLM Module → Response → Memory
